## 功能特性

### 核心功能
- **交互式色轮**：支持鼠标拖拽和触摸操作，实时选择颜色
- **双模式选择**：
  - **自由选择模式**：独立选择主色和次色，完全自由控制
  - **推荐选择模式**：选择主色后，系统自动推荐最佳配色方案

### 配色方案算法
基于经典色彩理论，提供 6 种配色方案：

| 配色方案 | 角度关系 | 特点 |
|---------|---------|------|
| **互补色** | 180° | 对比强烈，视觉冲击力大 |
| **类似色** | ±30° | 和谐统一，柔和舒适 |
| **三角色** | ±120° | 平衡且富有活力 |
| **分裂互补色** | 150°, 210° | 对比强烈但更加柔和 |
| **四角色** | 60°, 180°, 240° | 色彩丰富，适合复杂设计 |
| **单色系** | 相同色相 | 简洁优雅，层次分明 |

### 其他特性
- **实时预览**：即时显示所选颜色的渐变效果
- **HSL 信息显示**：显示当前颜色的色相、饱和度、明度值
- **颜色值输入**：支持直接输入十六进制颜色值
- **响应式设计**：适配桌面和移动设备
- **触摸支持**：支持移动设备触摸操作

## 色彩推荐算法

色彩推荐算法基于互补色原理：

1. 将用户选择的颜色从HEX格式转换为RGB格式
2. 将RGB转换为HSL（色相、饱和度、亮度）格式
3. 计算互补色相（色相值+180度）
4. 使用相同的饱和度和亮度生成互补色
5. 将互补色转换回HEX格式

## 使用方法

### 自由选择模式
1. 点击"选择主色"或"选择次色"按钮激活对应选择器
2. 在色轮上点击或拖拽选择颜色
3. 也可以直接在输入框中输入十六进制颜色值

### 推荐选择模式
1. 切换到"推荐选择模式"
2. 从下拉菜单选择配色方案类型
3. 在色轮上选择主色
4. 系统自动计算并显示推荐的配色
5. 点击推荐的颜色块可切换次色

## 色彩推荐算法详解

### 互补色 (Complementary)
```
推荐色相 = (主色相 + 180°) % 360°
```
色轮上相对的颜色，形成强烈对比。

### 类似色 (Analogous)
```
推荐色相1 = (主色相 + 30°) % 360°
推荐色相2 = (主色相 - 30° + 360°) % 360°
```
相邻的颜色，营造和谐统一的视觉效果。

### 三角色 (Triadic)
```
推荐色相1 = (主色相 + 120°) % 360°
推荐色相2 = (主色相 + 240°) % 360°
```
等边三角形的三个颜色，平衡且富有活力。

### 分裂互补色 (Split Complementary)
```
推荐色相1 = (主色相 + 150°) % 360°
推荐色相2 = (主色相 + 210°) % 360°
```
互补色两侧的颜色，对比强烈但更加柔和。

### 四角色 (Tetradic)
```
推荐色相1 = (主色相 + 60°) % 360°
推荐色相2 = (主色相 + 180°) % 360°
推荐色相3 = (主色相 + 240°) % 360°
```
矩形配色方案，色彩最丰富。

### 单色系 (Monochromatic)
```
相同色相，不同明度：[30%, 50%, 70%, 85%]
```
相同色相不同明度，简洁优雅。
