# 色彩选择器

这是一个使用 Vue 3 + Vite 构建的色彩选择工具，用于生成完美的渐变配色方案。

## 功能特点

### 🎨 色轮选择
- 在单个色轮上同时选择两种颜色
- 支持鼠标和触摸操作
- 实时显示选中颜色的 HEX 值

### 🔄 两种选择模式
1. **自由选择模式**：用户可以在色轮上自由选择任意两种颜色
2. **推荐选择模式**：选择第一个颜色后，系统自动推荐最佳配色方案

### 🎯 色彩推荐算法
基于经典的色彩理论，提供以下推荐方案：
- **补色**：色轮上相对的颜色，高对比度，视觉冲击力强
- **类似色**：色轮上相邻的颜色，和谐统一
- **分裂补色**：补色两侧的颜色，平衡对比度与和谐
- **三角色**：色轮上等距的三种颜色，丰富多样
- **四角色**：色轮上等距的四种颜色，饱满平衡

### ✨ 其他功能
- 随机配色生成
- 一键复制渐变 CSS
- 实时渐变预览
- 响应式设计

## 使用说明

1. **选择模式**：点击顶部按钮切换"自由选择模式"或"推荐选择模式"
2. **选择颜色**：在色轮上点击或拖动选择颜色
3. **应用推荐**（推荐模式）：选择颜色 1 后，点击推荐的配色方案
4. **随机配色**：点击"随机配色"按钮获得随机配色方案
5. **复制渐变**：满意后点击"复制渐变"按钮复制 CSS 代码

## 技术栈

- **Vue 3** - 渐进式 JavaScript 框架
- **Vite** - 新一代前端构建工具
- **Vitest** - 新一代前端测试框架
- **Canvas API** - 用于绘制色轮

## 色彩理论参考

本项目实现了以下经典色彩搭配理论：
- 补色搭配 (Complementary)
- 类似色搭配 (Analogous)
- 分裂补色搭配 (Split Complementary)
- 三角色搭配 (Triadic)
- 四角色搭配 (Tetradic)
